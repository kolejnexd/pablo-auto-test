# Autohandel & Abschleppdienst Pablo e.U.

Projekt Next.js 14 (App Router) z Tailwind CSS i TypeScriptem dla strony firmy Autohandel & Abschleppdienst Pablo e.U.

## Wymagania
- Node.js 18+
- npm lub pnpm

## Instalacja
```bash
npm install
```

## Tryb deweloperski
```bash
npm run dev
```
Aplikacja uruchomi się domyślnie na http://localhost:3000.

## Budowanie produkcyjne
```bash
npm run build
```
Po zbudowaniu aplikację można uruchomić komendą `npm start`.

## Komendy używane w testach

```bash
npm install
npm run lint
npm run build
npm run dev
```
